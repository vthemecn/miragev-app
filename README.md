MirageV APP
-------

### 项目介绍
MirageV 主题配套小程序，本代码使用 uni-app 开发，一套代码，可发布到iOS、Android、Web（响应式）、以及各种小程序、快应用等多个平台。

前端基于uni-app开发，后端基于WordPress开发，运行此项目，需要安装WordPress，并安装 MirageV。
